# ✅ Добавлена функция деактивации пользователей

## Что добавлено

В форму "Изменить пользователя" добавлена кнопка **"Деактивировать пользователя"**.

### Как это работает:

1. Владелец выбирает пользователя для редактирования
2. Видит две кнопки:
   - **"Сохранить параметры"** (синяя) - сохраняет изменения
   - **"Деактивировать пользователя"** (красная) - деактивирует пользователя

3. При нажатии на "Деактивировать пользователя":
   - Показывается подтверждение: "Деактивировать этого пользователя? Он не сможет войти в систему."
   - Если ОК → обновляется `is_active = false` в базе данных
   - Пользователь больше не может войти в систему
   - Форма очищается автоматически

### Защита:

✅ Нельзя деактивировать самого себя (показывается ошибка)  
✅ Требуется подтверждение перед деактивацией  
✅ Деактивированный пользователь автоматически не может войти (проверка в `fetchEmployeeForUser`)

## Как использовать

### Сценарий 1: Деактивировать пользователя

1. Войдите как владелец (owner)
2. Перейдите в "Управление пользователями"
3. Нажмите "Изменить пользователя"
4. Выберите пользователя из списка
5. Нажмите кнопку "Деактивировать пользователя"
6. Подтвердите действие
7. ✅ Пользователь деактивирован

### Сценарий 2: Попытка деактивировать себя

1. Выберите свой собственный аккаунт
2. Нажмите "Деактивировать пользователя"
3. ❌ Показывается ошибка: "Вы не можете деактивировать свой собственный аккаунт."

### Сценарий 3: Деактивированный пользователь пытается войти

1. Пользователь был деактивирован (is_active = false)
2. Пытается войти с правильным email/паролем
3. ❌ Показывается ошибка: "Доступ запрещён. Обратитесь к администратору."

## Реактивация пользователя

Если нужно реактивировать пользователя:

### Через Supabase Dashboard:

1. Откройте Supabase Dashboard → Table Editor → employees
2. Найдите пользователя
3. Измените `is_active` с `false` на `true`
4. ✅ Пользователь снова может войти

### Или добавьте кнопку "Реактивировать" (если нужно):

В будущем можно добавить аналогичную кнопку для реактивации деактивированных пользователей.

## Стиль кнопки деактивации

Кнопка имеет **красный цвет** для предупреждения:
- Фон: Красный градиент (прозрачный)
- Рамка: Красная
- Текст: Светло-красный
- Эффект: Привлекает внимание к опасному действию
